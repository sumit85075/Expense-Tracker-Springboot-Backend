Expense Tracker Spring Boot Backend

A robust backend REST API for a personal finance and expense tracking application built using Spring Boot Spring Data JPA and MySQL with Spring Security token based API authentication and authorization

Features
-> User Authentication Token based security JWT
-> Role based Authorization Secure endpoints for Users and Admins
-> Expense Management CRUD operations for tracking expenses and categories
-> Data Persistence Relational database mapping using Spring Data JPA

Tech Stack
-> Backend Java Spring Boot
-> Security Spring Security JWT JSON Web Token
-> Database MySQL
-> ORM Hibernate Spring Data JPA
